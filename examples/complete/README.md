# Complete Example

This example demonstrates the complete usage of the terraform-alicloud-ros module, showing how to create and manage ROS (Resource Orchestration Service) resources including stacks, templates, stack groups, and related components.

## Usage

To run this example, you need to execute:

```bash
$ terraform init
$ terraform plan
$ terraform apply
```

Note that this example may create resources which cost money. Run `terraform destroy` when you don't need these resources.

## Cost Estimation

The resources created in this example may incur costs. Please refer to the [Alibaba Cloud Pricing](https://www.alibabacloud.com/pricing) for details.
